== Description ==

This is a very lightweight Buddypress plugin that list maximum 5 newest members in the widget area

== Installation ==

1. Install the plugin through the 'Plugins >> Add New' menu in WordPress or do a manual upload.

2. Activate the plugin

3. Go to your widget area and look for "Recent Member"
